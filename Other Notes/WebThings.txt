SEO - Search Engine Optimization
--------------------------------
-Meta title 
	create your title with proper name with web name and page name ( 12Pro - IPhone)
-Meta description
	<meta name="description" content="the text you are placing here is used as a tag for SEO">
-Meta robots
	<meta name=”robots” content="index, follow"> 
	-an HTML snippet that tells search engine robots what they can and cannot do on a certain page
-Meta refresh redirect
	-Using meta refresh to create an instant client-side redirect
	<meta http-equiv="refresh" content="5;url="https://site.com/">
-Meta charset
	<meta http-equiv="Content-Type" content="text/html;charset=UTF-8">
	-UTF-8 - Character encoding for Unicode;
	-ISO-8859-1 - Character encoding for the Latin alphabet.
-Meta viewport
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	- Shows SEO to the site is compitable with different devices

Web 1.0
-------

-> 1985 to 2005
-> Type of Web : Visual or text-based Web
-> first stage of the World Wide Web evolution
-> Basic Web Pages - Static Web Pages 
-> Uses Pure HTML
-> website is only readable.
-> Not Supports Two Way Communication.


-----------------------------------2️⃣--------------------------------
Web 2.0
-------

-> 1999 to 2012
-> Introduction to Web Applications with Different Functionalities.
-> Type of web : Interactive, Responsive, and, Programmable Web
-> highlight user-generated content
-> Supports Multiple Scripting Languages.
-> website is readable as well user can create it.
-> introduced different functionalities with APIs and Tools due to supports libraries.
-> Good Introduction to Two Way Communication
-> Different Device Accessablity.
-> Success to High Speed Communication and data transfer.
-> World Wide Connectivity through different Protocols and
